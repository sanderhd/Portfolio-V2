Quiz App

The Quiz App is a web application that allows users to take quizzes on various topics. It is built using PHP and provides a user-friendly interface for creating, taking, and managing quizzes.

Features

  • Create and manage quizzes
  • Take quizzes and get instant feedback
  • View quiz results and statistics
  • User authentication and profile management

Technologies Used

  • PHP
  • HTML & CSS
  • JavaScript
  • MySQL

GitHub Repository

You can find the source code for the Quiz App on GitHub: https://github.com/sanderhd/Ctrl-Alt-Elite
