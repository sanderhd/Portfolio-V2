Blog Posts

This Blog Post site is built using React. It allows users to create, edit, and delete blog posts. The site is designed to be responsive and user-friendly, providing a seamless experience for users on both desktop and mobile devices.

Features

  • Create, edit, and delete blog posts
  • Responsive design

Future Plans/Work in Progress

  • Commenting system

Technologies Used

  • React
  • Tailwind CSS
